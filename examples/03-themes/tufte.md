# Tufte Theme

Inspired by Edward Tufte's book designs.

## Features

The Tufte theme emphasizes:

- Wide margins for annotations
- Elegant serif typography
- Minimal chrome and decoration

## Code Example

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```
