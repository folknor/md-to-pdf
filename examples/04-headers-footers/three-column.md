# Three-Column Layout

This example demonstrates headers and footers with left, center, and right columns.

## Configuration

```yaml
header:
  left: "{title}"
  right: "{date}"
footer:
  left: Confidential
  center: Page {page} of {pages}
  right: Draft v1.0
```

## Use Cases

Three-column layouts are ideal for:

- Professional reports
- Legal documents
- Technical specifications
- Corporate communications
